# react-horizontal-stacked-bar-chart-example

React horizontal stacked bar responsive chart component examples.

## Usage

- clone this project

```bash
git clone https://github.com/ricardodorosario/react-horizontal-stacked-bar-chart-example.git
```

- start this project

```bash
npm install
npm start
```
